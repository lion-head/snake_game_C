Snake Game (C Language)

Welcome to my Snake Game project, crafted solely by me.
Experience the classic gameplay using arrow keys and alphabetic keys to control the snake.
Seamlessly pause and resume the game, and compete with friends for the highest score.
Immerse yourself in the world of snakes with this C language-powered gaming experience.

  • Features :

    1. Classic Snake Game: Experience the timeless fun of the Snake Game.
    2. C Language Implementation: Developed entirely in C language.
    3. Arrow Key Controls: Control the snake's movements using arrow keys.
    4. Alphabetic Key Controls: Unique twist allowing control with alphabetic keys.
    5. Pause and Resume: Seamlessly pause the game and resume whenever desired.
    6. Save Functionality: Save your game progress to continue from where you left off.
    7. High Score Tracking: Compete with others by aiming for the highest score.
    8. Smooth Exit: Exit the game smoothly and effortlessly.
    9. Single-Player Gameplay: Enjoy the game solo and challenge yourself.
